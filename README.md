# Odyssey Map

An interactive infographic map of Odysseus's 10-year journey from Homer's *The Odyssey*, presented in chronological order.

![Odyssey Map](https://jasonycw.github.io/odyssey-map/)

## Features

- **Interactive Map**: Leaflet.js-powered map with dark cartographic tiles
- **14 Chronological Stops**: From Troy to Ithaca, with detailed descriptions
- **Animated Journey Path**: The route draws itself on page load
- **Auto-Play Mode**: Sit back and watch the journey unfold
- **Timeline Navigation**: Jump to any stop via the timeline bar
- **Responsive Design**: Works on desktop and mobile
- **Rich Content**: Each stop includes book references, dramatic descriptions, and historical context

## Usage

Visit the live site at: [https://jasonycw.github.io/odyssey-map/](https://jasonycw.github.io/odyssey-map/)

Or clone and open `index.html` locally (requires internet for map tiles and fonts).

## Technology

- Single HTML file with embedded CSS and JavaScript
- [Leaflet.js](https://leafletjs.com/) for interactive mapping
- CartoDB dark_matter tiles via OpenStreetMap
- Google Fonts (Cinzel + Cormorant Garamond)
